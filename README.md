# Classic Checkers

A modern, responsive Checkers (Draughts) game built with React, TypeScript, and Tailwind CSS.

## Features

- **Game Modes**:
  - **PvP**: Play against a friend on the same device.
  - **vs AI**: Challenge the computer with 3 difficulty levels (Easy, Medium, Hard).
- **Game Logic**:
  - Valid move highlighting.
  - Mandatory jumps (force capture).
  - Multi-jump support (double/triple jumps).
  - King promotion.
  - Win detection.
- **UI/UX**:
  - Clean, responsive design using Tailwind CSS.
  - Smooth animations and visual feedback.
  - Collapsible settings menu.

## Tech Stack

- **Frontend**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
